anna-app LLM Demo

Exercises anna.llm.complete (direct), anna.tools.invoke against an Executa that calls sampling/createMessage (single-shot and an N-step sample_chain that exercises max_calls + sampling-token renewal), and the full agent.session surface (create/run/cancel/history/refresh/delete/list) over both the HOST API and the Reverse RPC transports, including the session lifecycle (sliding idle window, token-free release, resume-by-uuid after reload) and the runtime-accurate tool surface (granted_tools / inherit_host_tools on create + the run_meta first frame with its NO_TOOLS_AVAILABLE warning).
